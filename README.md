# Waist Joystick Controller

## 프로젝트 소개

허리 움직임을 이용해 차량을 제어하는 IoT 프로젝트입니다.

실제 구현에서는 부품 대여 문제로 자이로 센서 대신 조이스틱을 사용하여 시스템을 구현했습니다.
